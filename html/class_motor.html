Detailed Description

Motor class derived from Component.

This class uses the inherited polygon (mImage) from Component as the shaft. It also has an additional Polygon member (mBody) for the motor body. The Draw function is overridden so that it upcalls the base Component draw (drawing the shaft) and then draws the motor body.
